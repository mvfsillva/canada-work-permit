### 01. Install dependencies

```sh
$ yarn
```

### 02. Create configs

Create a `.env` file at the root of the project. Make sure you follow the [`.env.example`](.env.example) file as a guide.

### 03. Run the tests

```sh
$ yarn test
```

### 03. Run the project
```sh
$ yarn dev
```

### 03. Run the storybook
```sh
$ yarn storybook
```
